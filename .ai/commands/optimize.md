# Optimize Command

## Usage
`/optimize <target>`

## Context
Use this command to analyze and improve performance of a specific component or endpoint.

## Steps
1. **Profile**: Analyze current performance (bottleneck identification).
2. **Propose**: Suggest improvements (e.g., streaming, caching, indexing).
3. **Verify Safety**: Ensure no behavioral changes (Optimization Rules).
4. **Implement**: Apply optimizations.
5. **Verify**: Measure improvement.
